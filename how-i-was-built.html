Key Engineering Decisions

⚠️ Azure CDN Classic was retired

The Cloud Resume Challenge originally specified Azure CDN. In 2025, Microsoft retired Azure CDN classic and no longer accepts new profile creation. The official replacement (Azure Front Door) costs ~$35/month. We chose Cloudflare Free as a cost-effective alternative providing the same CDN, HTTPS, and DNS capabilities.

⚠️ Host Header mismatch with Cloudflare + Azure Storage

Cloudflare's proxy sends the custom domain (vincentlelaverda.com) as the HTTP Host header to Azure Storage. Azure Storage static website only accepts its own hostname and returns InvalidUri 400. A Cloudflare Worker intercepts each request and rewrites the Host header to the correct Azure Storage endpoint before forwarding.

✅ Terraform over ARM templates

The challenge suggests ARM templates for IaC. We used Terraform instead — it is cloud-agnostic, more readable, and manages both Azure and Cloudflare resources in the same codebase with a unified state file.
